# 快速排序
`快速排序`是一种高效的通用排序算法。快速排序是一种分而治之的算法。它的工作原理是从数组中选择一个`基准`元素，然后根据它们是小于还是大于`基准`将其他元素分成两个子数组。这可以就地完成，需要少量额外的内存来执行排序。
